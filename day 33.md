# Media Integrity Analyzer

**Media Integrity Analyzer** is a single-file interactive web application designed to teach media literacy through guided discovery instead of traditional quizzes.

The application encourages users to slow down, observe carefully, identify persuasive techniques, and understand how headlines, emotional language, and audience targeting influence the way information is consumed.

## Key Features

* Interactive Headline Detective
* Emotion Detection Challenge
* Live Media Integrity Metrics
* Personalized Learning Dashboard
* Replayable scenarios
* Premium editorial-inspired UI
* Responsive design
* Built entirely with HTML, CSS, and Vanilla JavaScript
* Fully offline with no external dependencies

## Why I Built This

Modern software can do more than automate tasks—it can help people develop better thinking habits.

This project explores how interactive design can make abstract concepts like media bias, emotional persuasion, and critical thinking easier to understand through hands-on exploration.

## What I Learned

* Designing educational user experiences
* Structuring guided learning flows
* Creating reusable UI components using Vanilla JavaScript
* Building responsive layouts without external frameworks
* Balancing visual design with accessibility and usability
