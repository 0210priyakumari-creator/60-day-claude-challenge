# Day 20 – Face Puzzle Game

## Overview

Today, I developed a Face Puzzle Game using HTML, CSS, and JavaScript. The application captures a user's photo through the webcam, converts it into puzzle pieces, and challenges the user to solve the scrambled image.

## Features

* Webcam access using `getUserMedia()`
* Photo capture functionality
* Multiple difficulty levels (3×3, 4×4, 5×5)
* Puzzle generation from captured image
* Drag-and-drop support
* Touch gesture support for mobile devices
* Real-time timer and move counter
* Correct piece tracking
* Win detection system
* LocalStorage-based leaderboard
* Responsive user interface

## Skills Practiced

* JavaScript DOM Manipulation
* Canvas API
* Image Processing
* Event Handling
* Drag and Drop Mechanics
* Touch Events
* Local Storage Management
* Responsive Web Design

## Challenges Faced

* Managing drag-and-drop interactions smoothly across devices
* Ensuring puzzle pieces snap correctly into grid positions
* Creating a responsive layout that works on both desktop and mobile screens
* Implementing accurate timer and move tracking

## Key Learning Outcomes

* Improved understanding of browser camera APIs
* Enhanced knowledge of image slicing using Canvas
* Better handling of user interactions through mouse and touch events
* Experience building a complete interactive game without external frameworks
