## 🚀 Operation Lifeline: Supply Chain Crisis Lab

Operation Lifeline is a premium single-page React simulation that transforms supply chain management into an interactive learning experience.

### Features

* 🎮 Interactive crisis management simulator
* 🏢 Random company generation
* ⚠️ Dynamic supply chain disruption scenarios
* 🤝 Supplier negotiation system
* 👔 CEO leadership decision assessment
* 🤖 AI strategy investment planning
* 📊 Live business KPI dashboard
* 📈 Performance analytics and personalized feedback
* 🌙 Premium enterprise-inspired dark UI
* 📱 Fully responsive design
* 🔄 New randomized experience every playthrough

### Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* React (CDN)
* Babel JSX
