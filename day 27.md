# Day 27 — Prior Authorization Story Simulator 💙🏥

Today's project focuses on simplifying one of the most complex healthcare administrative processes—**Prior Authorization (PA)**—through an interactive, story-driven simulator.

## 🚀 What I Built

An educational web application using:

* HTML
* Tailwind CSS
* Vanilla JavaScript

## ✨ Features

* 📖 8-scene interactive healthcare story
* 💬 Chat-based conversation interface
* 👦 Patient (Rahul) and 👧 Healthcare Operations Specialist (Priya)
* 📈 Progress tracker for each workflow stage
* 🎯 Decision-based branching after every scene
* 📚 Beginner-friendly explanations
* 🏥 Real-world Prior Authorization workflow visualization

## 🩺 Story Covers

* Doctor Visit
* Prior Authorization Submission
* Insurance Review
* Step Therapy
* Denial
* Appeal Process
* Final Approval
* Healthcare System Metrics

## 💡 Key Learnings

This project helped me understand that healthcare administration is not just paperwork—it directly impacts patient care, treatment timelines, provider workload, and insurance decision-making.

Building educational simulations is a powerful way to transform complicated workflows into engaging learning experiences.
