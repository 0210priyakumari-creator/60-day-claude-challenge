# 📄 PDF Splitter & Merger

Day XX of my AI & Web Development Challenge 🚀

A premium single-page PDF utility designed with a modern UI/UX that enables users to split and merge PDF files entirely in the browser without uploading documents to any server.

## ✨ Features

- 📑 Split PDFs by page numbers or custom ranges
- 🔀 Merge multiple PDF files into one
- 🖼️ Page thumbnail previews
- 📋 Drag-and-drop file uploads
- ↕️ Reorder PDFs before merging
- 📊 Live page count and output summary
- 🌙 Dark mode interface
- ⚡ Fast client-side processing
- 🔒 100% private—files never leave your device
- 📱 Fully responsive design
- ♿ Accessibility-friendly interface
- 🎨 Smooth animations and modern UI

## 🛠️ Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Client-side PDF libraries

## 🎯 Learning Outcomes

- Advanced UI/UX Design
- Client-side document processing
- Drag-and-drop interactions
- PDF manipulation in the browser
- Responsive web development
- Performance optimization
- Accessibility best practices

Building practical tools like this strengthens both frontend development and user experience design while keeping user privacy at the core.
