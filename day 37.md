# 🚀 Task Compass

**Task Compass** is a single-file offline educational web application designed to teach how work flows through real organizations.

Instead of testing job titles, the application helps users understand how teams collaborate to solve real workplace problems.

## ✨ Features

* Three-stage management simulation
* Interactive ownership-based decision making
* Drag-and-drop role assignment
* Realistic workplace scenarios
* Progress tracking
* Detailed explanations after every decision
* Fully responsive interface
* Works completely offline

## 🛠️ Built With

* HTML5
* CSS3
* Vanilla JavaScript

No frameworks, external libraries, APIs, or backend services are required.

## 🎯 Learning Outcomes

* Ownership
* Delegation
* Collaboration
* Escalation
* Organizational workflows
* Team responsibilities

This project focuses on helping learners think like managers and understand how tasks move across different roles inside modern organizations.
